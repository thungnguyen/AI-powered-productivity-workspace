# ZenNote

**ZenNote** is an AI-powered productivity workspace inspired by Notion, TickTick, and Linear. It is designed for personal productivity and engineered to scale into a real SaaS product.

## Tech Stack
- **Backend:** ASP.NET Core Web API, Clean Architecture, CQRS, MediatR
- **Database:** MySQL, Redis
- **Real-time:** SignalR
- **Frontend:** Flutter
- **Infrastructure:** Docker, GitHub Actions

## Repository Structure
- `.github/`: CI/CD pipelines and GitHub templates.
- `architecture/`: ADRs, architecture diagrams, and system designs.
- `backend/`: ASP.NET Core Web API.
- `database/`: Database schemas, seed data, and migration configurations.
- `docker/`: Dockerfiles and docker-compose orchestration.
- `docs/`: Developer guides and project documentation.
- `frontend/`: Flutter mobile and web application.
- `prompts/`: AI system prompts and configurations.
- `rules/`: Project linting, formatting rules, and Git hooks.
- `scripts/`: Automation, build, and environment scripts.
- `tasks/`: Task tracking and sprint planning.
